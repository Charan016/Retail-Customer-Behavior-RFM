# Retail Customer Behavior Analysis (RFM)

## 📌 Overview

This project analyzes customer purchasing behavior using **RFM (Recency, Frequency, Monetary)** segmentation and **Market Basket Analysis** to identify valuable customer groups and product associations.

## 🎯 Objectives

* Segment customers based on purchasing behavior
* Identify high-value and at-risk customers
* Discover frequently bought-together products

## 🧰 Tech Stack

* Python (Pandas, NumPy, Scikit-learn)
* Jupyter Notebook
* SQL (data cleaning)
* Power BI (dashboard)

## 📊 Key Steps

1. Data Cleaning & Preprocessing
2. RFM Feature Engineering
3. Customer Segmentation
4. Market Basket Analysis (Apriori)
5. Visualization & Dashboard

## 📁 Files

* `Online_Retail_RFM_Project.ipynb` – main analysis
* `rfm_segments.csv` – customer segments
* `market_basket_rules.csv` – association rules
* `FlowChart.jpeg` – workflow
* `retail_dashboard.pbix` – Power BI dashboard

## 🚀 Outcome

* Identified high-value customers for targeted marketing
* Found strong product associations for cross-selling
* Built insights for business decision-making
